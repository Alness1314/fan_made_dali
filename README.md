# BTS Landing Fan

Landing page estatica creada con React, Vite y Tailwind CSS.

## Ejecutar

```bash
npm install
npm run dev
```

Para generar la version de produccion:

```bash
npm run build
```

## Personalizacion

- URL y nombre de Instagram: `CONFIG` en `src/App.jsx`
- Frase inferior: `CONFIG.quote` en `src/App.jsx`
- Imagen de fondo: `src/assets/img/bts-concert.jpeg`
- Logo de Instagram: `src/assets/img/instagram.svg`
- Simbolo inferior: `src/assets/img/bts-symbol.svg`
- Audio ambiental: `src/assets/sound/purple-ambient.wav`

Los imports de imagenes y audio estan agrupados al inicio de `src/App.jsx`.
